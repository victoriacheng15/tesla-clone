## Run Locally

Clone the project

```bash
  git clone git@github.com:victoriacheng15/tesla-clone.git
```

Go to the project directory

```bash
  cd tesla-clone
```

Install dependencies

```bash
  npm install
```

or 

```bash
  yarn
```

Start the server

```bash
  npm run dev
```
or 

```bash
  yarn dev
```
